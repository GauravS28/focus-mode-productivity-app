# Focus Mode Productivity App

A modern productivity application designed to help users stay focused, manage tasks efficiently, and improve daily productivity through time management and focus tracking.

## Live Demo

https://focus-mode-productivity-app.vercel.app/

## Features

* Focus timer for distraction-free work sessions
* Task management and productivity tracking
* Daily progress monitoring
* Clean and responsive user interface
* Focus session analytics
* Productivity insights and statistics
* Mobile-friendly design

## Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/GauravS28/focus-mode-productivity-app.git
cd focus-mode-productivity-app
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
