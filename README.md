# Focus Mode – Productivity Web App

Focus Mode is a minimal productivity web application designed to help users stay focused, manage tasks, and improve concentration during work or study sessions.

## Features

- Pomodoro-style focus timer
- Start / Pause / Reset controls
- Clean and responsive UI
- Background focus ambience
- Task tracking support
- Lightweight and fast performance

## Tech Stack

- HTML5
- CSS3
- JavaScript
